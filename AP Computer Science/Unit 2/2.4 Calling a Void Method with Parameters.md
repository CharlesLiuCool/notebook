**Void** method just means that the program doesn't return a value.

```java
public class Calculator {
	public void calcAverage(int num1, double num2) { //method signature
		double average = (num1 + num2) / 2;
		System.out.println("average: " + average);
	}
	
	//note I can overload operators, just like in C++
	//this is where I have same name but different parameters
	public void calcAverage(int num1, double num2, double num3) { //method signature
		double average = (num1 + num2 + num_3) / 3;
		System.out.println("average: " + average);
	}
	public static void main(String[] args) {
	Calculator avg = new Calculator();
	avg.calcAverage(200,0.3);
	avg.calcAverage(200, 100, 10.5);
	}
}
```
